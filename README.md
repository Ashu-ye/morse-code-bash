# Morse Code Converter (Bash)

A lightweight terminal-based tool to encode plain text into Morse code and decode Morse signals back into English.

## Features
- **Encode:** Converts alphanumeric text to Morse.
- **Decode:** Translates Morse signals (using `/` for spaces).
- **Executable:** Can be run as a standalone script in any Bash environment.

## How to Run
1. Give execution permissions:
   ```bash
   chmod +x morse.sh

2.  Run the script:

```bash
   ./morse.sh
